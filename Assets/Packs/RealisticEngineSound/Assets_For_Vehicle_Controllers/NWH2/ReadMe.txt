If you want to use Realistic Engine Sounds with NWH2 Vehicle Physics,
import NWH2 Vehicle Physics, Realistic Engine Sounds and NWH2_RES-Plus.unitypackage into your project.
This unitypackage contain demo scenes, scripts for demo scenes and prefabs for NWH2 Vehicle Physics.
This prefabs are ready to use with NWH2 Vehicle Physics.

Get NWH2 Vehicle Physics here: https://assetstore.unity.com/packages/tools/physics/nwh-vehicle-physics-2-166252?aid=1101l3r5B

Read the documentation for tutorials.
