If you want to use Realistic Engine Sounds with RCK(Realistic Car Kit),
import Realistic Car Kit, Realistic Engine Sounds and later RCK_RES-Plus.unitypackage.
This unitypackage contain demo scenes, scripts for demo scenes and prefabs for Realistic Car Kit. This prefabs are ready to use with Realistic Car Kit vehicles.
If you don't own Realistic Car Kit, you can buy it here: https://assetstore.unity.com/packages/tools/physics/realistic-car-kit-18421?aid=1101l3r5B

Read the documentation for tutorials.
