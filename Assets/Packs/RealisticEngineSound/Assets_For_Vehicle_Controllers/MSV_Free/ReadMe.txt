If you want to use Realistic Engine Sounds with MS Vehicle System - Free,
import MS Vehicle System - Free, Realistic Engine Sounds and later MSV-Free_RES-Plus.unitypackage.
This unitypackage contain demo scenes, scripts for demo scenes and prefabs for MS Vehicle System. This prefabs are ready to use with MS Vehicle System vehicles.
If you don't own MS Vehicle System, you can get it here: https://assetstore.unity.com/packages/tools/physics/ms-vehicle-system-free-version-90214?aid=1101l3r5B

Read the documentation for tutorials.
