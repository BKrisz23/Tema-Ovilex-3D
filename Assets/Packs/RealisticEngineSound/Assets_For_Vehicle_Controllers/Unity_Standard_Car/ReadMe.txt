If you want to use Realistic Engine Sounds with Unity Standard Assets,
import Unity Standard Assets, Realistic Engine Sounds and UnityCar_RES-Plus.unitypackage.
This unitypackage contain demo scenes, scripts for demo scenes and prefabs for Unity Standard Assets' Car. This prefabs are ready to use with Unity Standard Assets' Car.
Get Unity Standard Assets here: https://assetstore.unity.com/packages/essentials/asset-packs/standard-assets-32351?aid=1101l3r5B
