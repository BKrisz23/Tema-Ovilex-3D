If you want to use Realistic Engine Sounds with MSK(Motorbike Starter Kit),
import Motorbike Starter Kit, Realistic Engine Sounds and later MSK_RES-Plus.unitypackage.
This unitypackage contain demo scenes, scripts for demo scenes and prefabs for Motorbike Starter Kit. This prefabs are ready to use with Motorbike Starter Kit.
If you don't own Motorbike Starter Kit, you can buy it here: https://assetstore.unity.com/packages/templates/motorbike-starter-kit-19156?aid=1101l3r5B
Get Motorbike Engine Sounds - RES Addon Pack here: https://goo.gl/LPkoZy This package contain 11 motorbike engine sound packages, prefabs and demo scenes for both RES and MSK.
